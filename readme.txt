# Zemní práce Pávek - Website

Professional excavation and earthworks company website.

## Setup Instructions

1. Upload all files to your web server
2. Ensure PHP mail() function is enabled on your server
3. Create a `logs` directory with write permissions (755)
4. Add your machinery and project photos to respective image folders
5. Update calendar password in `jscalendar.js` (line 2)
